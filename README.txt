BEFORE USING
1. Install requirements.txt
2. Dowload google credential_json file and save into key\ with name is credential.json

USING

1. Save photo into Resources\Photos

2. Run colorPicker.py and edit HSV range, HSV range will auto save to hsv_saved.csv
Run again if like to reset to default

3. Run main.py
	Auto trim word image
	Delete wrong image in Resources\Trimmed_Images
	Click OK => Auto create concanated image in Resources\Concatenate_Images
	Check concanated image and edit using Pain
4. Run google_vision.py to save word list into Results
